# Glyco Demo Script

1. Open Overview and show risk, monitoring status, flags, and weekly summary.
2. Open Risk Check and run the default profile assessment.
3. Open Monitoring, add a glucose reading, choose fasting or not fasting, and show the updated trend state.
4. Open Reports and generate doctor, family, and weekly reports.
5. Open Agent, ask "Should I be worried this week?", and show tool calls, guideline grounding, safety note, and Agent Memory.
6. Use "Teach Glyco" to save feedback with a preferred tone, then ask again and show learned focus, recent glucose pattern, and next-best action.
7. Open Care Plan and show practical nutrition and activity guidance.
8. Open Family View and show the simplified read-only support dashboard.
